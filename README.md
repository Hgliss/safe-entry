SafeEntry
Sistema seguro de registro de entradas y salidas para guarderías
Descripción del proyecto

SafeEntry es un sistema digital desarrollado como proyecto freelance para una guardería infantil, con el objetivo de mejorar la seguridad y el control en el proceso de registro de entradas y salidas de los niños.

El sistema reemplaza los registros manuales tradicionales por una plataforma digital segura, permitiendo que únicamente los padres o tutores autorizados puedan retirar a los menores. Además, el sistema mantiene un historial confiable de accesos y permite a la administración monitorear los registros en tiempo real.

La solución fue diseñada pensando en seguridad, control, trazabilidad y facilidad de uso para el personal de la guardería y los padres de familia.

Problema que resuelve

En muchas guarderías el control de ingreso y salida de los niños se realiza de forma manual, utilizando cuadernos o listas físicas. Este método puede generar diversos problemas como:

Falta de verificación sobre quién retira al menor

Registros incompletos o poco confiables

Dificultad para consultar historiales de entradas y salidas

Falta de notificaciones inmediatas para los padres

SafeEntry fue desarrollado para digitalizar y asegurar este proceso, reduciendo errores humanos y aumentando la seguridad dentro del centro infantil.

Solución propuesta

SafeEntry implementa un sistema de control de accesos que permite registrar de forma segura la entrada y salida de los niños mediante códigos QR y verificación mediante PIN.

El sistema registra cada evento en tiempo real y permite que los administradores del centro infantil mantengan un control detallado de todas las actividades relacionadas con el acceso.

Funcionalidades principales
Gestión de usuarios

El sistema permite registrar diferentes tipos de usuarios con distintos niveles de acceso:

Administrador

Personal administrativo

Docentes

Padres o tutores

Cada rol cuenta con permisos específicos dentro del sistema.

Autenticación segura

El sistema incluye mecanismos de seguridad para garantizar el acceso adecuado a la plataforma:

Inicio de sesión seguro

Control de acceso basado en roles

Verificación mediante PIN para acciones sensibles

Registro mediante código QR

Cada niño cuenta con un código QR único que permite registrar:

Entrada al centro infantil

Salida del centro infantil

Verificación del tutor autorizado

Esto ayuda a garantizar que el menor sea entregado únicamente a personas autorizadas.

Notificaciones en tiempo real

El sistema permite enviar notificaciones a los padres cuando:

El niño ingresa a la guardería

El niño es retirado

Se registra algún evento importante

Historial de accesos

SafeEntry mantiene un registro digital completo de entradas y salidas, lo que permite:

Consultar movimientos diarios

Revisar historiales

Mejorar el control administrativo
